# Carbon Kids Site

Welcome to the Carbon Kids Site! This website is designed to educate children aged 5-14 about carbon footprints and how they can help the environment. Through fun facts, interactive games, and engaging activities, we aim to inspire the next generation to take action against climate change.

## Project Structure

The project is organized as follows:

```
carbon-kids-site
├── src
│   ├── index.html          # Main entry point for the website
│   ├── pages               # Contains individual pages of the site
│   │   ├── home.html       # Home page with fun facts and visuals
│   │   ├── about.html      # Information about carbon footprints
│   │   ├── learn.html      # Educational resources for kids
│   │   └── activities.html  # List of activities and games
│   ├── components          # Reusable components for the site
│   │   ├── header.html     # Header component with navigation
│   │   ├── footer.html     # Footer component with copyright info
│   │   └── card.html       # Card component for displaying facts/activities
│   ├── styles              # CSS styles for the site
│   │   ├── main.css        # Main styles for layout and typography
│   │   ├── variables.css    # CSS variables for consistent design
│   │   └── components.css   # Styles for reusable components
│   ├── js                  # JavaScript files for functionality
│   │   ├── main.js         # Main JavaScript functionality
│   │   ├── games.js        # Interactive games related to carbon footprints
│   │   └── utils.js        # Utility functions for common tasks
│   └── data                # JSON data files
│       ├── facts.json      # Fun facts about carbon footprints
│       └── activities.json  # List of activities to reduce carbon footprints
├── package.json            # npm configuration file
├── .gitignore              # Files to ignore in version control
└── README.md               # Documentation for the project
```

## Getting Started

To get started with the Carbon Kids Site, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd carbon-kids-site
   ```

3. **Install dependencies**:
   ```
   npm install
   ```

4. **Run the project**:
   You can open `src/index.html` in your web browser to view the site.

## Purpose

The purpose of this website is to make learning about carbon footprints fun and engaging for children. By providing interactive content and activities, we hope to empower kids to make a difference in their environment.
